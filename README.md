# 📊 Task 03 - Decision Tree Classifier

## 🎯 Objective
To develop a Decision Tree model to predict whether a customer will subscribe to a term deposit.

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Google Colab  

## 📌 Steps Performed
- Imported and loaded the dataset  
- Examined column names and dataset structure  
- Converted categorical variables into numeric values  
- Selected input features (X) and target variable (y)  
- Split data into training and testing sets  
- Trained the Decision Tree Classifier  
- Evaluated performance using accuracy and confusion matrix  
- Plotted the decision tree for visualization  

## 💡 Key Insights
- Decision Tree helps identify customer behavior patterns  
- Model provides clear decision rules  
- Accuracy indicates overall model performance  
- Some features have strong impact on subscription prediction
